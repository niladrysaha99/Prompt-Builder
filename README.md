# ⬡ PromptForge — AI Image Prompt Builder

A beautiful, fully client-side prompt builder for AI image generation tools like **Midjourney**, **DALL·E**, **Stable Diffusion**, and **Flux**.

![PromptForge Screenshot](screenshot.png)

## ✨ Features

- 🎨 **Tag-based builder** — click to add styles, moods, lighting, camera angles & quality boosters
- 🚀 **Quick Presets** — 6 ready-made prompt templates (Fantasy, Portrait, Sci-Fi, Nature, Abstract, Architecture)
- 🔀 **Multi-platform support** — outputs optimized prompts for Midjourney, DALL·E, Stable Diffusion & Flux
- 📋 **One-click copy** — instantly copy your prompt to clipboard
- 📊 **Live stats** — real-time character & word count
- 🌙 **Dark theme** — easy on the eyes

## 🚀 Deploy to GitHub Pages (Free)

1. **Fork or clone this repo**
   ```bash
   git clone https://github.com/YOUR_USERNAME/prompt-builder.git
   cd prompt-builder
   ```

2. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit"
   git push origin main
   ```

3. **Enable GitHub Pages**
   - Go to your repo → **Settings** → **Pages**
   - Under **Source**, select `main` branch → `/ (root)`
   - Click **Save**

4. **Your site is live at:**
   ```
   https://YOUR_USERNAME.github.io/prompt-builder/
   ```
